五、新增全局属性（了解）

学习阁大约 1 分钟languageHTMLlanguageHTML前端
属性名功能
contenteditable表示元素是否可被用户编辑，可选值如下：
true ：可编辑
false ：不可编辑
draggable表示元素可以被拖动，可选值如下：
true ：可拖动
false ：不可拖动
hidden隐藏元素
spellcheck规定是否对元素进行拼写和语法检查，可选值如下：
true ：检查
false ：不检查
contextmenu规定元素的上下文菜单，在用户鼠标右键点击元素时显示。
data-*用于存储页面的私有定制数据。

示例：

演示
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>新增的全局属性</title>
    <style>
        div {
            width: 600px;
            height: 200px;
            border: 1px solid black;
            font-size: 20px;
            margin-bottom: 10px;
        }
        .box1 {
            background-color: skyblue;
        }
        .box2 {
            background-color: green;
        }
    </style>
</head>
<body>
    <div class="box1" spellcheck="true" contenteditable="true">Lorem ipsum dolor sit amet.</div>
    <div class="box2" draggable="true" hidden data-a="1" data-b="2" data-c="3">Lorem ipsum dolor sit amet.</div>
    <h1>hello</h1>
    
</body>
</html>
